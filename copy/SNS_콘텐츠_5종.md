## 4. 다이브 체험 — "5분이면 한 편의 이야기"

### 플랫폼: 인스타 릴스 / 틱톡
### 포맷: 세로 영상 9:16 (20~30초)

**영상 스토리보드:**
```
[0-3초]  어두운 화면, 물방울 사운드
         텍스트: "5분, 한 편의 이야기"

[3-7초]  폰 화면 목업 — 루시브 선택 화면
         손가락이 "잠들지 못한 도시" 탭

[7-12초] 다이브 시작 — 채팅 UI
         LP: "이 도시에는 비밀이 있어."
         선택지 2개 등장

[12-16초] 선택지 A 탭 → 스토리 분기
          텍스트 빠르게 흘러감 (타임랩스)

[16-20초] 다이브 완료 화면
          "히든 엔딩 달성!" + 별점 ★★★★★
          LP: "이 결말, 나도 처음이야."

[20-24초] 공유 화면 — 랭킹 #3 달성

[24-27초] Lucive 로고
          "이야기 속으로, 다이브."
```

**캡션:**
```
5분이면 하나의 세계를 경험할 수 있어요.

AI가 만든 이야기 속 주인공이 되어
선택하고, 분기하고, 결말을 바꾸세요.

히든 엔딩을 찾으면... 👀

#Lucive #5분다이브 #AI스토리 #인터랙티브
#히든엔딩 #선택지게임 #비주얼노벨
```
