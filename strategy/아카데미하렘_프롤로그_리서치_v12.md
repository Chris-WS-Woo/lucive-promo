## Part 1 · 웹툰 프롤로그 레퍼런스 (7)

아카데미/회귀/하렘 장르에서 **첫 화가 강렬하다**고 알려진 작품들과 그들의 훅 패턴.

### 1. 학사재생 (Academy's Genius Swordmaster)
- **프롤로그 훅**: 폐허가 된 왕도에서 주인공이 피투성이로 죽음 → 눈을 뜨니 17살 아카데미 입학 전야. "이번에는 다르게."
- **비주얼**: 다크 퍼플/블루 + 실버 검 광택 + 금발 남주 + 시네마틱 렌즈 플레어
- **캐릭터 소개 방식**: 1화 끝 컷에 4명의 히로인 실루엣이 순차 페이드 (은발 황녀 → 금발 공녀 → 흑발 광녀 → 백발 신관)
- **시그니처**: *"같은 세계에서, 다시 시작한다."*
- **우리가 훔칠 것**: **회귀 훅** — LP 오픈에 "어제 있었던 일" 암시

### 2. 악녀는 모래시계를 되돌린다 (로판 회귀)
- **프롤로그 훅**: 대리석 위 피, 왕의 조소, 처형 직전 — 모래시계가 뒤집힌다
- **비주얼**: 크림 골드 + 와인 레드 + 로즈골드 레이스 장식
- **캐릭터 소개**: 여주 혼자 먼저 + 이후 챕터별 히로인/악역들 등장
- **시그니처**: *"다시는 그들을 믿지 않겠어."*
- **우리가 훔칠 것**: **장식적 아트** (프레임·봉인·꽃 모티프) — LP hero에 차용

### 3. 아카데미에 집착광공이 너무 많다
- **프롤로그 훅**: 여주 입학식 → 4명의 남자가 동시에 눈길을 멈춘다 (각자 다른 집착 유형: 얀데레/쿨/피폐/순정)
- **비주얼**: 파스텔 + 로맨스 글로우 + 캐릭터 눈 하이라이트
- **캐릭터 소개**: **4컷 그리드** — 동시에 4명을 한 페이지에
- **시그니처**: *"이 아카데미, 뭔가 이상해."*
- **우리가 훔칠 것**: **4컷 히로인 동시 소개** — LP v12의 핵심 beat

### 4. 아카데미의 천재 검사
- **프롤로그 훅**: 스승의 죽음 회상 → 묘비 앞 맹세 → 아카데미 입학
- **비주얼**: 다크 네이비 + 은빛 검 + 폐허 배경
- **캐릭터 소개**: 라이벌·스승·동기가 번갈아 등장하는 몽타주 컷
- **우리가 훔칠 것**: **몽타주 컷 시퀀스** — 여러 캐릭터를 짧은 샷으로 빠르게 플래시

### 5. 악역 영애가 되어 버렸다 (빙의물)
- **프롤로그 훅**: 책을 읽던 주인공이 눈을 뜨니 "처형당하는 악역"이 되어 있음
- **비주얼**: 크림 + 골드 + 장미 + 웨딩드레스 질감
- **시그니처**: *"살고 싶다."*
- **우리가 훔칠 것**: **메타 내레이션** — "당신이 읽고 있는 이 이야기의 주인공"

### 6. 나 혼자만 레벨업 (타 장르이지만 프롤로그가 최강)
- **프롤로그 훅**: 약한 헌터가 지하 던전에서 몰살 직전 → 각성 → 세계가 바뀐다
- **비주얼**: **보랏빛 오라 + 골드 스탯창 + 시네마틱 슬로모션**
- **우리가 훔칠 것**: **슬로모션 클로즈업** — 표정 하나가 트레일러의 터닝포인트

### 7. 내 남편과 결혼해줘 (회귀 로맨스)
- **프롤로그 훅**: 배신당한 주인공의 죽음 → 회귀 → 복수를 결심한다
- **비주얼**: 쿨 네이비 + 피처럼 붉은 립스틱 + 얼음 같은 눈빛
- **우리가 훔칠 것**: **눈빛 한 컷의 임팩트** — 대사 없이 표정만으로 스토리 전달

## Part 2 · Stardive UI 패턴 (Netmarble 게임 LP 표준)

Stardive 는 SPA라 DOM 스크레이핑이 한계. 네비 구조와 업계 표준으로 핵심 패턴을 정리.

### 2.1 Nav 구조 (확인됨)
`메인 / 캐릭터 / 세계관 / PV / 드롭스 / 미디어 / 크리에이터 / 웹상점`

→ LP v12에는 아래 5개 차용: **메인 · 캐릭터 · 세계관 · PV(티저) · 사전등록**

### 2.2 섹션별 전형적 구조 (업계 공통)

| Section | 구현 |
|---|---|
| **Hero** | 풀블리드 시네마틱 (정적 아트 + 파티클 + 큰 로고 + "사전 등록" CTA) |
| **PV (Promotion Video)** | 풀폭 비디오 플레이어 · 큰 ▶ 버튼 · thumbnail은 캐릭터 합성샷 |
| **캐릭터** | 가로 캐러셀 or 4컷 그리드 · 각 카드 클릭 시 확대 모달 (풀샷 + 프로필 + 시그니처 대사) |
| **세계관** | 지도/일러스트 + 세로 스토리텔링 (한 장면당 스크롤 한 스냅) |
| **드롭스** | 특별 캠페인 / 콜라보 섹션 (우리는 "알파 테스터 특전" 으로 대체) |
| **미디어** | 프레스 로고 · 트레일러 · 티저 이미지 |
| **커뮤니티** | 디스코드/트위터/유튜브 링크 |
| **사전등록 CTA** | 맨 아래 고정 + 상단 네비에 중복 · 카운트 (N명 등록 중) · 보상 계단 |

### 2.3 시네마틱 "트레일러" 장치 (Stardive 스타일)

- **밤하늘 파티클** (별, 깃털, 꽃잎) — hero에 지속
- **캐릭터 풀샷 등장 시 카메라 페이드** (하단에서 위로 슬라이드)
- **골드/네온 라인** 강조 — 프레임 경계
- **문자 타이핑 효과** (로고/슬로건만 제한적)
- **BGM 자동 재생** (머트 기본, 하단 ♪ 버튼)
- **카운트다운 타이머** (런칭까지 D-N)
- **실시간 사전등록 카운터** (1,243,812명 등록 중)

### 2.4 우리가 훔칠 것 (TOP 1)
→ **"캐릭터 4컷 그리드 → 호버/클릭 시 풀샷 모달"**

Stardive의 캐릭터 섹션이 가장 강한 포인트. 각 캐릭터 클릭 시 풀스크린 풀샷 + 시그니처 대사 + 음성이 재생되는 경험. 우리 4 히로인에 이걸 그대로 적용.

## Part 4 · 스크롤 · 인터랙션 상세

### 진행 방식 (Stardive 하이브리드)
- **풀스크린 세로 스크롤** + 각 Beat = 100vh
- **Beat 3–6 (히로인)**: 각 Beat 안에서 좌→우 슬라이드 전환 시 **다음 히로인** (키보드 ← → 또는 스와이프)
- **Beat 7**: 2×2 그리드, hover 시 해당 quadrant 확대 + 내부 대사 재생
- 모든 히로인 Beat에 **▶ TTS 버튼** (Kanna / Annie / 미정 × 2 voices로 pre-generate)

### BGM
- 3단계: Beat 1 (웅장 오프닝) → Beat 3–6 (각 히로인 톤) → Beat 7–8 (고조·결의)
- 각 Beat 트리거 시 cross-fade

### 파티클·이펙트
- Hero: 하늘에서 떨어지는 꽃잎 (느린)
- 히로인 Beat: 각자 고유 파티클 (아리아=별빛 / 니아=카메라 플래시 / 세하=책장 먼지 / 루나=눈송이)
- Final CTA: 골드 별빛
