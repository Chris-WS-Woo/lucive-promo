## 1. 프로젝트 세팅 (2분)

### 새 프로젝트 생성

1. CapCut 실행 → `New Project`
2. 우측 상단 `Canvas` 클릭
3. **Custom → 1080 × 1920** 입력 (9:16 세로, 원본 기준)
4. Frame rate: **30 fps** (모든 플랫폼 호환)
5. 프로젝트명: `LUCIVE_VN_DEMO_V1`

> **팁:** 9:16을 원본으로 만들고 나중에 16:9·1:1로 크롭하는 게 쉬움. 반대로 하면 손해.

### 내보내기 설정 미리 저장

`Export` → 기본값 변경:
- 해상도: 1080p
- 프레임 레이트: 30
- 품질: 고화질 (Higher)
- 코덱: H.264
- 비트레이트: 8-12 Mbps

## 2. 30초 영상 — 장면별 구성 (핵심)

> **원칙:** 5개 컨셉이 각 4-6초씩 드러나도록 설계. 랜딩 페이지 v4-final.html의 타임라인 오버레이와 **정확히 동기화**.

```

00:11 ─── 00:17  [VISUAL NOVEL]   씬 전환 + 캐릭터 표정 + 선택지
```

**CapCut 작업:**
1. `Background` 패널 → Aurora 그라디언트 이미지 업로드 (Figma에서 1080x1920 export)
2. Text → "Lucive", 폰트 Inter 900, 120pt, Fill 색상 #F1F3F8

## 4. 한국어 자막 설정 (필수)

### 자막 폰트·크기

```
폰트: Noto Sans KR (CapCut 한국어 기본 폰트)
Weight: Bold (700) 또는 Black (900)
Size: 52pt (9:16 기준)
Fill: #FFFFFF (white)
Stroke: #000000 2px (가독성)
Shadow: 검정 70% opacity, 오프셋 0,4, blur 8
```

### 자막 위치
- **하단 1/4 영역**에 배치 (0.75 Y 기준)
- 안전 영역: 상하좌우 80px 여백
- 틱톡·인스타 UI가 가리지 않도록 과도하게 하단 X

### 자막 줄바꿈 규칙
- 한 줄 최대 **15자** (모바일 기준)
- 한 자막당 최대 **2줄**
- 표시 시간: 한 글자당 0.15초 (자막 30자면 4.5초 유지)

### 자막 AI 자동 생성

CapCut → `Text` → `Auto Captions`:
1. 한국어 선택
2. 자동으로 음성 인식하여 자막 생성
3. 오타만 수동 교정
4. 스타일 일괄 변경: 모든 자막 선택 → 폰트/색상 수정

## 5. 🎁 One Source → Many Outputs — 파생 제작

> 30초 원본이 완성되면 여기서 **7-10개 파생물**을 뽑아냅니다.

### 🅰️ 15초 숏폼 3종 (TikTok·Reels·Shorts)

같은 프로젝트를 복제해서 **3가지 버전**으로 컷:

**버전 1 — MEMORY 집중 (0:05-0:20 선택)**
- 30초 원본에서 Scene 2 + 3 + 4만 잘라냄
- 인트로 1초 + 핵심 12초 + CTA 2초 = 15초
- 훅 타이틀: "AI가 17일 전 대화를 기억한다"

**버전 2 — VISUAL NOVEL 집중 (Scene 3 + 4 + 5)**
- 훅 타이틀: "장면이 움직이는 AI 대화"

**버전 3 — CHOICE 집중 (Scene 4 확장)**
- 훅 타이틀: "숨겨진 선택지가 히든 엔딩을 연다"

**작업:**
1. 프로젝트 복제 (`File → Save As`)
2. 타임라인에서 불필요한 씬 삭제
3. 인트로·CTA 축소
4. 내보내기

### 🅱️ 9초 GIF (디시·아카용)

가장 임팩트 있는 9초 구간 선택 → GIF 변환:

**구간 추천:**
- Scene 2 (MEMORY) 5-11초 → 6초 + CTA 3초 = 9초
- 또는 Scene 3-4 (VN + CHOICE) 11-20초 중 9초

**CapCut → Export → Format: GIF**
- 또는 MP4 → [ezgif.com](https://ezgif.com/video-to-gif) 변환
- 목표 파일 크기: **800KB 이하**

### 🅲️ 정사각(1:1) 버전 (X·인스타 피드)

1. 프로젝트 복제
2. `Canvas` → **1080 × 1080**로 변경
3. 타임라인 클립 선택 → `Crop` → 중앙 정사각으로 재프레이밍
4. 자막 위치 중앙 하단으로 재배치
5. 내보내기

### 🅳️ 가로(16:9) 버전 (X 가로·블로그)

1. 프로젝트 복제
2. `Canvas` → **1920 × 1080**로 변경
3. 9:16 씬을 중앙 세로 바에 두고 좌우를 **흐린 블러 배경**으로 채움
4. 또는 씬을 완전 재편집 (로고·캐릭터 재배치)

### 🅴️ 스틸 이미지 5장

영상 재생 중 베스트 프레임 추출:
1. 타임라인에서 원하는 순간으로 이동
2. `Preview` 창 우측 상단 → `Export Frame`
3. 1080×1920 PNG 저장
4. 추천 추출 포인트:
   - 0:03 (로고 풀샷)
   - 0:09 (메모리 증명 채팅)
   - 0:15 (VN 씬 + 대사창)
   - 0:20 (히든 선택지 강조)
   - 0:29 (CTA + 카운트다운)

## 6. 내보내기 — 채널별 최적 설정

### 랜딩 페이지용 (v4-final.html `assets/vn-demo.mp4`)

```
Format: MP4 (H.264)
Resolution: 1080 × 1920
FPS: 30
Bitrate: 8 Mbps
Audio: AAC 128 kbps
목표 크기: 5-8 MB
```

**추가 최적화:** [Handbrake](https://handbrake.fr/) 돌려서 Web Optimized 체크 후 재인코딩 → 2-3MB로 압축 가능.

### TikTok (1080×1920, 9:16)
- MP4 1080p 30fps, 9 Mbps, ≤ 287MB
- 15-60초 권장

### Instagram Reels (1080×1920, 9:16)
- MP4 1080p 30fps, 8 Mbps
- 15-30초 권장

### YouTube Shorts (1080×1920, 9:16)
- MP4 1080p 30fps, 8 Mbps
- 15-60초 권장

### X(트위터) (1:1 정사각)
- MP4 1080×1080 30fps, 5 Mbps
- ≤ 140초, ≤ 512MB

### GIF (디시·아카)
- 480×852 (축소), 15fps, ≤ 800KB
- 팔레트: 256색

## 11. 품질 업그레이드 팁 (2주차 이후 여유 있을 때)

### After Effects 수준 연출 CapCut으로

- **Speed Ramp:** 클립 선택 → `Speed` → `Curve` → 가속·감속 구간 설정 (임팩트 씬에 효과적)
- **Keyframe 애니메이션:** 위치·크기·투명도·회전 모두 키프레임 설정 가능
- **Motion Blur:** 빠른 움직임 클립에 `Effects` → `Motion Blur` 추가
- **Color Grading:** `Adjust` → `Curves`로 섀도우에 시안, 하이라이트에 퍼플 추가 (Lucive 브랜드 룩)
- **LUT 적용:** `Adjust` → `LUT` → 시네마틱 LUT 선택

### 유료 소스 보조 (여유 있으면)

- **Envato Elements** ($16/월) — 프리미엄 SFX·BGM·템플릿
- **Motion Array** — After Effects 템플릿 (고급 모션 그래픽)
- **Artgrid** — 시네마틱 B-roll 영상

## 13. 실전 예시 — 랜딩 페이지 오버레이 타이밍 매칭

v4-final.html의 `.demo-overlay data-start/end` 값과 영상 타임라인이 **정확히 맞아야** 합니다:

| 랜딩 오버레이 | 영상 씬 | data-start | data-end |
|--------------|---------|:----------:|:--------:|
| ▪ MEMORY | Scene 2 | 1 | 5 |
| ▪ VISUAL NOVEL | Scene 3 | 6 | 11 |
| ▪ CHOICE | Scene 4 | 12 | 16 |
| ▪ ORIGIN | Scene 5 | 17 | 22 |
| ▪ INITIATE | Scene 6 | 23 | 30 |

**주의:** 영상의 0초 인트로를 뺀 **5초부터 시작**으로 배치해야 오버레이가 정확히 뜸. 영상 인트로를 더 짧게(2초) 만들면 오버레이 값도 조정 필요.

> 영상 편집 완료 후 v4-final.html의 `data-start`·`data-end` 값을 **실제 영상 타임라인에 맞춰 재조정**.
