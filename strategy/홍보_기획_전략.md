# Lucive MVP 홍보 기획 전략

## 타겟
- 일반 유저 (10대 후반 ~ 30대 초반, 인터랙티브 픽션/웹소설/비주얼노벨 관심층)
- 페르소나: 서연(감정 몰입형), 민준(수집/전략형)

---

## API 역할 분담

| API | 역할 | 용도 |
|-----|------|------|
| **Grok** | 텍스트 + 이미지 + 영상 + TTS | 홍보 카피 생성, 이미지 생성, 짧은 영상 클립, 나레이션 음성 |
| **Nanobanana** | 고품질 이미지 생성/편집 | 키비주얼, SNS 썸네일, 앱스토어 스크린샷 스타일 이미지 |
| **Higgsfield** | 시네마틱 영상 생성 | 홍보 영상 (틱톡/릴스/숏츠용 15~60초) |

---

## 홍보 콘텐츠 구성

### Phase 1: SNS 콘텐츠 (5종)

| # | 콘텐츠 | 플랫폼 | 포맷 | 핵심 메시지 |
|---|--------|--------|------|------------|
| 1 | 티저 — "당신의 선택이 결말을 바꾼다" | 인스타/틱톡/X | 이미지 + 짧은 카피 | 인터랙티브 스토리 체험의 매력 |
| 2 | LP 소개 — "나만의 AI 동반자를 만나세요" | 인스타/틱톡 | 캐러셀/짧은 영상 | AI 파트너와 함께하는 모험 |
| 3 | L-Maker — "당신도 스토리 크리에이터" | 틱톡/X | 영상/GIF | 누구나 AI로 스토리를 만든다 |
| 4 | 다이브 체험 — "5분이면 한 편의 이야기" | 인스타/틱톡 | 영상 | 짧고 강렬한 다이브 체험 |
| 5 | 공유 랭킹 — "내 플레이가 화제의 중심" | X/인스타 | 이미지 + 카피 | 소셜 경쟁과 공유의 재미 |

### Phase 2: 홍보 영상 (2종)

| # | 영상 | 길이 | 용도 |
|---|------|------|------|
| 1 | 틱톡/릴스용 숏폼 | 15~30초 | SNS 광고, 바이럴 |
| 2 | 앱 소개 영상 | 45~60초 | 앱스토어, 유튜브, 랜딩페이지 |

## 제작 파이프라인

```
1. 카피 작성 (Grok 텍스트 생성 보조)
2. 키비주얼 이미지 생성 (Nanobanana Pro — 고품질 4K)
3. 영상 스크립트 작성
4. 나레이션 TTS 생성 (Grok Voice)
5. 영상 클립 생성 (Higgsfield — 시네마틱 프리셋)
6. 최종 편집 및 산출물 정리
```
