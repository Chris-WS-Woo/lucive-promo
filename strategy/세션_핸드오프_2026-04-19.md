## 3. 확정된 결정 사항

### 프로덕트 스펙 (2026-04-18 lucive-redesign main 동기화 완료)
- **알파 100명** (타겟 A 60 + B 40), D30 리텐션 ≥ 35%
- **1 Dip = $0.01 고정**, 알파 웰컴 2,000 Dip ($20) 지급
- LP 대화 1건 = 1 Dip 고정 (LP MVP 기획서 명시)
- 신규 개념: 생활 이벤트 스케줄 · LP 일기 · 복귀 의식 · VN 모드 LP별 TTS

### LP 디자인 · 톤
- 경쟁사 비교는 **4열 그리드** 또는 **unified(4문제→1대화)** 또는 **focus(1주제 1카드)** 중 선택
- **v8 실험본은 B2C 저격 광고 톤** (Apple Get a Mac 인격화 + T-Mobile 단일 페인 + Slack 자신감 참고)
- 사용 중인 경쟁사 저격 대상: Character.AI · Talkie · Replika · Chai · 크랙 · 제타
- 가격 Row는 **"실제 정책 변경 이력"** 사용 (크랙 화폐 3번 개편 · Replika 친밀모드 삭제 등)
- 플로팅 네비는 pill + text (아이콘 only 금지)
- Hero CTA의 vs 점프 버튼: "크랙·제타가 못한 4가지" (직설·자신감 · 앱 실명 지목)

### 이메일·레퍼럴 스택 (확정 아님, 방향성)
- **Prefinery Launch ($49/월) + Klaviyo Free** 조합 권장
- Supabase 초기 단계 **필요 없음** — Prefinery가 대신 담당
- 연결 방식은 **커스텀 폼 + Prefinery API** (v7 LP 브랜드 감성 유지)

### 전화번호 / SMS 방침 (논의 진행 중)
- **전화번호는 알파 단계 수집 보류** (수집 마찰 · PIPA/TCPA 규제 부담)
- **대신 커뮤니티 기반 푸시**로 대체:
  - 🇰🇷 KakaoTalk 오픈채팅
  - 🇯🇵 LINE 공식계정 + OpenChat (양방향은 베타부터, 알파는 브로드캐스트만)
  - 🇺🇸 Discord 서버
- 가입 완료 화면에 **"커뮤니티 참여 →" 버튼** 추가는 아직 미적용 (다음 세션 작업 항목)

### 일본 시장 특수 전략 (결정됨)
- **LINE 오픈챗은 외부 API 불가** → Official Account API 활용
- 팀 내 일본어 구사자 없음 → **GPT 번역 봇** 직접 개발 방향
- 봇 구조: 유저 JP → Webhook → GPT(KR 번역) → Discord 팀 → KR 답변 → GPT(JP 번역) → LINE Push
- 알파 단계 일본 20-30명은 Tally 폼 + DeepL 배치 번역으로 감당

### 피드백 수집 (알파용)
