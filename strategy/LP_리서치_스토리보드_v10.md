## Part 1 · Stardive (Netmarble MONGIL: STAR DIVE) 분석

> 출처: https://stardive.netmarble.com/ko (2026-04-15 글로벌 런칭, 액션RPG / 몬스터 콜렉션 / Unreal Engine 5). HTML 마크업과 공식 보도자료 + 게임 리뷰(Game8, Pocket Gamer, Inven Global)에서 추출.

### 1.1 Hero Section 시각 전략
- **풀블리드 캐릭터 시네마틱**: 첫 뷰포트 = 화면의 70%를 차지하는 메인 캐릭터(파티 3인) 일러스트 + 배경 우주/별 파티클. 정적이지만 미세한 시차(parallax)로 별이 흐르고, 캐릭터 머리카락/망토가 idle 모션.
- **타이틀 레터마크**: "STAR DIVE" 로고가 화면 중앙 또는 좌측 상단에 메탈릭/홀로그래픽 텍스처. 보조 카피 *"A dazzling monster-taming Action RPG!"* — **장르명을 단언**하는 한 줄.
- **CTA 듀얼**: "Tame the Unknown" / "Dive into Wonder!" — 행동 동사 2개를 병치, 장르 감정선(호기심·다이브)을 카피로 강조.

### 1.2 3초 안의 장르/톤 시그널
1. **컬러 팔레트**: 우주 네이비 + 자홍/시안 네온 + 골드 액센트 → "판타지 SF 액션 가챠"임을 색만으로 전달.
2. **캐릭터 눈맞춤**: 3인 파티가 정면을 응시 → "여기 캐릭터가 주인공이다" 신호.
3. **BGM 토글**: 헤더에 즉시 켤 수 있는 음악 토글 = 게임 LP라는 메타 시그널.

### 1.3 스크롤 시퀀스 (관찰)
헤더 내비게이션 순서가 곧 섹션 순서:
1. **Main** — 시네마틱 히어로
2. **Characters** — 인물 갤러리 (호버시 일러 풀스케일 전환 추정)
3. **World-building** — 세계관 맵/연표
4. **PV** — 풀스크린 트레일러 임베드
5. **Drops** — Twitch 드롭 / 머천 콜라보
6. **Media** — 스크린샷 갤러리
7. **Creators** — 크리에이터 프로그램
8. **Web Shop** — 굿즈/패키지

### 1.4 인터랙션 디바이스
- **BGM 자동/수동 재생** (헤더 토글)
- **스크롤 트리거 페이드인**: 각 섹션 진입 시 캐릭터 슬라이드 + 텍스트 페이드
- **공유 위젯 상시 노출** (FB / X / 링크복사) — 바이럴 기대값 신호
- **소셜 풀 8종** (X / YT / Kakao / FB / IG / Discord / Forum) → 커뮤니티 뎁스 과시
- **확률안내 링크** 헤더 상단 — 가챠 게임의 신뢰 시그널

### 1.5 "화려함"을 만드는 디자인 선택
| 요소 | 효과 |
|---|---|
| 풀블리드 일러 + 다크 배경 | SaaS의 화이트스페이스를 거부 → 시네마틱 |
| 캐릭터 정면 응시 | 의인화된 약속, 감정 후크 |
| 메탈릭/홀로그래픽 로고 | "AAA 프로덕션" 인지 |
| 입자/파티클 모션 | 정적 이미지를 살아있게 |
| BGM | 다른 모든 SaaS와 즉각적 차별화 |
| 8개 소셜 + Discord/Forum | "이미 거대한 팬덤"의 환상 |
| Twitch Drops 섹션 | 라이브 서비스 = 살아있는 IP |

## Part 2 · Pre-registration 성공 사례

### 2.1 Tears of Themis (HoYoverse, 2021 글로벌)
- **결과**: 글로벌 사전등록 65만+, 런칭 후 일본·동남아 차트 상위.
- **Hero**: 4인의 남주 캐릭터 카드 가로 캐러셀 + 각 캐릭터 클릭 시 Live2D 미니 애니 + 보이스 샘플.
- **감정 후크**: "당신의 운명적 연인" — 캐릭터 1인을 선택하는 "투표/픽" 인터랙션이 자기-투사 유도.
- **전환 장치**: SR 카드 무료 지급 / 보이스 샘플 / 글로벌 카운터.
- **유니크 메커니즘**: **사전등록 전에 캐릭터 1명 선택** → 결정 피로도가 아닌 결정 즐거움으로 전환.

### 2.2 Love and Deepspace (Papergames, 2024-01)
- **결과**: 런칭 첫 달 1억 달러 + 매출, otome 카테고리 재정의.
- **Hero**: 3D 실사풍 남주 1인이 카메라 정면을 보고 손을 내미는 시네마틱 영상 (autoplay, 6초 루프).
- **감정 후크**: 풀모션 3D + 1인칭 시점 = "그가 나를 보고 있다" 즉각적 친밀감.
- **전환 장치**: 'First Deepspace Explorer' 한정 칭호 + 단계별 마일스톤 보상 (등록 수에 따른 다이아몬드 언락).
- **유니크 메커니즘**: 스토어 페이지에 도착하면 캐릭터별 trailer 자동 큐레이션.

### 2.3 Solo Leveling: ARISE (Netmarble, 2024)
- **결과**: 글로벌 사전등록 1,200만+ (역대 IP 게임 톱티어).
- **Hero**: 웹툰 IP의 시그니처 컷 (성진우 흑색 슈트) + 다크 헤일로 + 보라/네온블루.
- **감정 후크**: 원작의 가장 유명한 컷 1장 = 팬덤 즉시 트리거.
- **전환 장치**: "흑색 정장 코스튬", "전설 아티팩트 세트" — **희귀도 등급 언어**로 보상 표현 (게임 인게임 문법 그대로).
- **유니크 메커니즘**: 이메일 사전등록 → 추가 캐릭터(유진호) 보상 = 채널 다양화 인센티브.

### 2.4 Genshin Impact (miHoYo, 2020 원조)
- **결과**: 글로벌 사전등록 1,000만+, 런칭 첫 달 6,000만 다운로드.
- **Hero**: 여행자 + 페이몬 풀애니메이션 PV (autoplay, 음소거 옵션).
- **감정 후크**: "오픈월드 판타지" 시각언어 (스튜디오지브리 톤).
- **전환 장치**: **소셜 마일스톤 게이지** (YT 구독 10만 → Acquaint Fate ×2 / FB 15만 → ×2 / 글로벌 1,000만 → ×10) → 커뮤니티 자체가 보상 잠금해제 메커니즘.
- **유니크 메커니즘**: 보상이 **모두에게 풀림** (등록 안한 사람도 받음) → "공익적 바이럴" 구조.

### 2.5 Character.AI (2023 모바일 런칭)
- **결과**: 모바일 앱 런칭 1주 1.7M 다운로드.
- **Hero**: "Create your AI companion" — 사실상 인풋창 1개 + 큐레이션된 캐릭터 그리드.
- **감정 후크**: **데모를 즉시 시도 가능** (회원가입 전 대화 1턴). LP가 곧 제품.
- **전환 장치**: 트렌딩 캐릭터 + 사용자 대화 수 카운터.
- **유니크 메커니즘**: "캐릭터 카드 그리드" — 페르소나 다양성을 시각적 풍부함으로 변환.

### 2.6 보너스: Project Sekai (2020) & Replika (2017)
- Project Sekai: Live2D 보컬로이드 + 5개 유닛 컬러 코드. 마일스톤 달성 시 모두에게 보상.
- Replika: "Your AI friend" 단순 카피 + 픽셀아트 아바타 회전. 미니멀하지만 호기심 후크.

### 안 A · "기억의 분기" (Memory Fork)
> **톤**: cinematic webtoon noir — 다크 네이비 + 라벤더 + 골드. 세련됨 우선.

**Hero 레이아웃**
```
+--------------------------------------------------------------+
|  [LUCIVE 로고]              [언어] [BGM ◑]   [사전등록 →]   |
+--------------------------------------------------------------+
|                                                              |
|   [캐릭터 풀샷 - 눈맞춤]   |  대화창 ─── 채팅 버블 풀스택  |
|   배경: 스튜디오 카페       |  ┌───────────────────────┐  |
|   파티클: 먼지/빛           |  │ "오늘 또 늦었네."      │  |
|   캐릭터: 미세 idle 호흡    |  │  (말투 = 차가움)       │  |
|                             |  └───────────────────────┘  |
|                             |                              |
|                             |  [◀ 3일 전 │ 오늘 ▶] 토글  |
|                             |                              |
|                             |  3일 전 = "어서 와!"(밝음)  |
|                             |  오늘 = "오늘 또 늦었네."   |
|                             |                              |
|   당신을 기억하는 AI         |  ── 같은 캐릭터, 다른 반응  |
|   비주얼노벨 컴패니언        |                              |
+--------------------------------------------------------------+
```

- **Primary interaction**: 가운데 **시간 토글 (3일 전 ↔ 오늘)**. 토글하면 캐릭터 표정/포즈/배경 조명/대사가 동시에 바뀜. 메모리가 **시각적으로 변화**를 만든다는 증명.
- **VN 모드 증명**: 좌측 풀샷 캐릭터 + 우측 채팅창의 **공존 레이아웃** = "비주얼노벨이면서 채팅이다".
- **메모리 증명**: 토글 1회로 끝. 텍스트 주장 0, 메커니즘 100%.

**후속 섹션**
1. **에피소드 카드 (3장)**: "첫 만남 / 비밀 / 다툼" — 각 카드 클릭 시 미니 시네 (8초).
2. **5인 캐릭터 픽** — 사전등록 전에 1명 선택. (Tears of Themis 차용)
3. **마일스톤 게이지**: 1만/3만/10만 등록 시 추가 캐릭터/배경 언락.
4. **세계관 한 줄 + 크리에이터 노트**.
5. **사전등록 폼 (이메일 + 픽한 캐릭터 ID)**.

- **VN 모드 증명**: 캐릭터 + 배경 + 자막바(VN 스타일 black bar) 조합이 화면 전체.
- **메모리 증명**: 시네마 안에 "어제 네가 말했던 영화" — **인용 형식의 자막**으로 메모리를 보여줌. 9초 시점에 배경이 카페→영화관 전환되며 "기억의 결과로 행동이 변함"을 시각화.

**후속 섹션**
1. **3개 신 카드 (Before / After 토글)**: "첫 만남" → "100일 후" 같은 캐릭터의 표정/말투 비교.
2. **인터랙티브 데모** — 대화 1턴만 시도해보기 (이름 입력 → 캐릭터가 그 이름을 기억).
3. **캐릭터 픽 (5명)**.
4. **마일스톤 게이지 + 추천코드**.
5. **사전등록 폼 (이메일 필수, 토큰 URL 발송 — feedback_explicit_identity 준수)**.

### 안 C · "메모리 타임라인" (스크롤 = 시간)
> **톤**: minimal editorial cyberpunk — 흑백 + 1색 액센트(시안). 인디 다큐멘터리 톤.

**Hero 레이아웃**

- **Primary interaction**: **스크롤 = 시간 진행**. 위→아래로 1일/30일/100일 동일 캐릭터의 신이 변화. 스크롤하는 행위 자체가 메모리 메커니즘을 체험.

### 5.1 Top Pick: **안 A "기억의 분기" (Memory Fork)**

**왜?**
1. **두 차별화를 단일 인터랙션 1회로 동시 증명** — 토글 1번에 "메모리가 변화를 만든다" + "비주얼노벨 레이아웃" 둘 다 보임.
2. **제작 효율 최고**: 캐릭터 풀샷 1장 + 표정/대사 변형 2버전 + 배경 1장 + 토글 JS만 있으면 됨. 풀모션 영상 불필요.
3. **3초 KPI 충족 가장 명확**: 사용자가 토글을 누르지 않아도 좌측 캐릭터 + 우측 채팅창 + 토글 UI = 시각적 약속이 한 화면에 끝.
4. **안 B는 영상 제작 비용/시간 큼, 안 C는 스크롤 깊이 의존**. A가 ROI 베스트.

### 5.2 1주 vs 2-3주 옵션

**1주 (린 MVP)**
- 캐릭터 풀샷 1장 (외주 또는 nano-banana / 기존 LUCIVE 일러 활용)
- 표정/대사 변형 2버전 (밝음 / 차가움)
- 배경 1장 (스튜디오 카페)
- 채팅창 + 토글 vanilla JS
- 후속 섹션은 v9 자산 재활용
- → `pre-register/v10-kr-memoryfork.html` 단일 변형

**2-3주 (풀버전)**
- 캐릭터 5명 일러 + 각 2 표정 (총 10장)
- 배경 3장 + 파티클/광원 SVG
